# CRISPR Golden Gate Tool (v1.0)

这是一个用于自动化设计 CRISPR Golden Gate 引物并模拟构建pDIRECT_21C载体的工具。VoytasLab Genome Engineering Toolkit目前无法使用。鉴于自己实验室一般只需要用到两个载体以及两个gRNA，方便大家使用。功能比较简陋，能力有限，目前够使用了。

## 功能特点
* 自动设计 6 条 Golden Gate 引物。
* 自动模拟 PCR 和酶切连接过程。
* 支持加载本地 PMOD 和 pDIRECT 模板文件。

## 如何下载
请点击右侧的 **"Releases"**，下载最新的 `.exe`或`.zip` 文件即可在 Windows或Mac 上直接运行，无需安装其他软件，可以直接打开使用。

## 如何使用
1. 下载并打开软件。
2. 加载您的 `PMOD.fasta` 和 `pDIRECT.fasta` 模板文件。
3. 输入 gRNA 序列。
4. 点击“生成结果”并保存 FASTA 文件。
